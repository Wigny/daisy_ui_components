# LiveDaisyuiComponents

DaisyUI + Liveview

Experimental project using `liveview 0.18`.

## Installation

Reference this repository on your `mix.exs` file to start using.

```elixir
def deps do
  [
    {:live_daisyui_components, github: "phcurado/live_daisyui_components"}
  ]
end
```
